**Domain:**
- Online English language learning platform  
- Vocabulary building & retention tools 

**Scope**  
- Personal dictionary integration  
- Flashcards for active recall & spaced repetition  
- Video examples showing words in real speech (from series/shows)

**Goal**  
- Help learners better memorize and retain new vocabulary  
- Improve learning outcomes and test results  

**Business Value**  
- Strengthen competitive position (feature parity with market leaders)  
- Increase user engagement and retention through effective tools  
- Support core educational process: learn â†’ practice â†’ consolidate

**Key Pain Points**     
- Need quick, intuitive way to mark & repeat difficult words  
- Desire for contextual examples (audio + video) beyond plain translation  
- Long loading times frustrate users  

**Solution:**
- Formalize user interactions via Use Cases  
- Align stakeholder expectations  
- Provide structured input for development  
- Support functional requirements elaboration
